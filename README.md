# אמירים - ניוזלטר

זהו אתר פשוט לניוזלטר של אמירים.

## מה יש באתר
- דף HTML יחיד
- עיצוב פשוט ומרכזי
- מתאים להצגה ישירה ב-GitHub Pages או בשרת סטטי

## איך לצפות באתר
פתח את הקובץ [index.html](index.html) בדפדפן, או הרץ שרת מקומי עם:

```bash
python3 -m http.server 8000
```

ואז פתח בכתובת:

```text
http://localhost:8000
```

## איך לעדכן
1. ערוך את [index.html](index.html)
2. שמור את השינויים
3. Commit ו-push ל-GitHub

## פריסה
האתר מתפרסם דרך GitHub Pages מהענף main של המאגר.
