# האתר של ליאב — עיתוני הסרטים המצוירים

אתר אינטראקטיבי בעברית סביב עיתוני סרטים מצוירים, הכולל אוסף משחקים, פרופילים אישיים, לוחות תוצאות והגדרות התאמה אישית.

## מה יש באתר
- דף HTML יחיד (`index.html`) שמכיל את כל האתר — עיצוב, קוד ותוכן
- מגוון משחקים: מילה מעורבבת, נחש מה בתמונה, כותרת חתוכה, סדר את הכותרות, מי אמר את זה, מצא את השגיאה, נחש לפי צבע, אמת או שקר, חדשות מזויפות, תשבץ עיתוני, נחש את הסרט, מה הציטוט ועוד
- כלים יצירתיים: מחולל כתבות, עורך עיתון אישי, כתב/עורך ראשי, מחולל ניחוש עתיד, צ'אט עם דמות
- מצבים שונים: מצב 2 שחקנים, מצב מהירות, מצב קשה, אתגר יומי, אתגר שבועי
- פרופיל אישי, כרטיס ביקור, אוסף כרטיסיות, מד ידע ולוח תוצאות
- כניסת אדמין עם יומן פעילות
- הגדרות נרחבות: שפת ממשק, סגנון עיצוב, נגישות, אפקטים, קול ומוסיקה, מצב תצוגה (יום/לילה) ושמירה

## איך לצפות באתר
פתח את הקובץ [index.html](index.html) בדפדפן, או הרץ שרת מקומי עם:

```bash
python3 -m http.server 8000
```

ואז פתח בכתובת:

```text
http://localhost:8000
```

## איך לעדכן
1. ערוך את [index.html](index.html)
2. שמור את השינויים
3. Commit ו-push ל-GitHub
